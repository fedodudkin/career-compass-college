# IT Roadmap — Дорожные карты IT-специальностей

## Краткое описание
Веб-сервис с интерактивными roadmap по IT-специальностям колледжа. Студент может выбрать свою специальность, просмотреть список навыков и отмечать пройденные темы.

## Цель сервиса
Помочь студентам IT-специальностей структурировать своё обучение: понять, что нужно изучить, видеть прогресс и не теряться в потоке информации.

## Инструкция запуска

### Вариант 1 — Live Server (рекомендуется)
1. Установи расширение **Live Server** в VS Code
2. Открой папку проекта в VS Code
3. ПКМ на `index.html` → **Open with Live Server**

### Вариант 2 — Python HTTP Server
```bash
cd college-roadmap
python -m http.server 8000
# открой http://localhost:8000
```

### Вариант 3 — GitHub Pages
Проект задеплоен по адресу: `https://[username].github.io/college-roadmap`

> ⚠️ Открывать `index.html` напрямую через файловую систему **не рекомендуется** — `fetch()` для загрузки `data.json` требует HTTP-сервер.

## Реализованный функционал

### Страницы
| Файл | Описание |
|------|----------|
| `index.html` | Главная — карточки специальностей с прогрессом |
| `about.html` | О сервисе — описание возможностей |
| `roadmap.html` | Основная — интерактивный roadmap |
| `form.html` | Форма обратной связи |

### JavaScript-функционал
- **Динамическая загрузка данных** из `data/data.json`
- **Отметка прогресса** — клик по навыку переключает статус
- **Сохранение в LocalStorage** — прогресс не теряется при перезагрузке
- **Фильтрация** по специальности на странице Roadmap
- **Сворачивание** секций специальностей
- **Форма с валидацией** — проверка всех полей, счётчик символов, сохранение заявок в LocalStorage
- **Адаптивное меню** — burger-меню на мобильных

## Структура проекта
```
college-roadmap/
├── index.html          # Главная страница
├── about.html          # Описание сервиса
├── roadmap.html        # Основная страница с roadmap
├── form.html           # Форма заявки
├── README.md
├── css/
│   └── style.css       # Все стили, CSS-переменные, адаптив
├── js/
│   ├── main.js         # Навигация, общий функционал
│   ├── index.js        # Логика главной страницы
│   ├── roadmap.js      # Логика roadmap (фильтры, прогресс)
│   └── form.js         # Валидация и отправка формы
└── data/
    └── data.json       # База данных специальностей и навыков
```

## Технологии
- **HTML5** — семантическая разметка
- **CSS3** — Flexbox, Grid, CSS-переменные, анимации
- **JavaScript** — ванильный JS (ES6+), без фреймворков
- **LocalStorage** — хранение прогресса и заявок
- **Git / GitHub Pages** — версионирование и деплой
